Різновидність велосипедів

Тип велосипеду Опис
Дорожній велосипед Знайомий, напевно, кожному. Рама, зазвичай, виконана зі сталі, що робить цей велосипед важким та неповоротким. Плюсом даного типу є те, що він порівняно з іншими велосипедами дешевий.
Міський велосипед Є технічним продовженням застарівших дорожніх велосипедів. Рама може бути виконана як зі сталі, так й з алюмінію. Дані моделі обладнані багажником, болотниками, широким сідлом, захистом ланцюга та інколи мають передню й задню фари. На відміну від попередньго виду, ці велосипеди є багатошвидкісні, що суттєво полегшує їзду та робить її більш комфортною.
Гірський велосипед (MTB) Найбільш поширений вид велосипедів, оскільки є універсальним для їзди по асфальту та бездоріжжю. Під поняттям «горний велосипед» розрізняють декілька видів, що дуже відрізняються між собою.
  • Гірський велосипед початкового рівня – це недорогі моделі з навісним обладнанням низької якості. Може бути обладнаний дешевою амортизаційною вилкою, а також різноманітними велоаксесуарами, типу велопідножки, широкого та м’якого сідла, багажника. По суті являється тим же прогулянковим велосипедом, лише з виду схожий на гірський. Область застосування та сама – катання по місту та околицях.
  • Гірський велосипед для кросс-кантрі – пересічна місцевість, багаточисленні спуски та затяжні підйоми – звичне діло для даного типу. Рама виконана з легких сплавів, професійне навісне обладнання, гідравлічні гальма, вилки, що мають довгий хід – характерні особливості цих моделей.
  • Велосипеди для downhill – призначені для швидкісних спусків з гір з природними та штучними перешкодами, а отже мають важку і міцну раму, потужні дискові гальма, а також амортизатори з ходом 200 мм.
Шосейний велосипед Велосипед, який призначений для швидкої їзди по рівному асфальті. Обладнаний надлегкою рамою, має максимально низьку посадку для зменшення опору вітру та дуже тонкі колеса. Впізнається за рахунок характерного руля (баранячий ріг). Рух по пересічній місцевості, розбитій дорозі та стрибки з трамплінів абсолютно протипоказані для даного типу.
Гібрид Поєднує в собі все найкраще від прогулянкового, гірського та шосейного велосипеда. Великі колеса підтримують високу швидкість, а амортизована вилка дозволяє пересуватись по нерівній поверхні. Ідеальний варіант для тих, хто обожнює швидку їзду по місту, але не проти проїхатись і лісовою стежкою.
BMX Велосипед, призначений для виконання трюків. Має 20-дюймові колеса, спеціальні підставки по бокам колес – пегі, а також механізм, що дозволяє обертати руль в одному напрямку довільну кількість разів.
Фікс Той же трековий велосипед, але пристосований для міської їзди. Гальма зазвичай також відсутні. Для зупинки використовується гальмування скідом - заднє колесо блокується зусиллям опору ніг на педалі, і велосипед йде юзом.
Трековий велосипед Призначений для гонок по велотреці. На даних моделях є всього одна фіксована передача. Педалі не мають вільного ходу відносно заднього колеса, а тому неперервно обертаються. Гальмами дані моделі не облаштовуються, гонщики зупиняються за рахунок поступового уповільнення ходу. Спроможні розвивати швидкість до 90 км/год.
